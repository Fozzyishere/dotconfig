# How to use

1. Copying .config, .local and .themes into your $HOME directory
2. Use nwg-looks to change GTK themes, fonts and icons for GTK apps
3. Use qt5ct/qt6ct to change QT themes, fonts and icons respectively
4. ...
5. Profit!
